# Music Hall Static

Web app to explore an artist's albums and music tracks. [Based on](https://github.com/allen-garvey/music-hall), but without the requirement on Vue or node.

## Dependencies

* go >= 1.24
* bun >= 1.3.3
* make

## Getting Started

* Clone or download this repository
* `make`
